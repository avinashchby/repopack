# repopack

Pack an entire codebase into a single LLM-optimized markdown file — ready to paste into any AI chat or API call.

## Quick Start
```bash
npx @avinashchby/repopack
```

## What It Does

Sharing a codebase with an LLM requires manually copying files or writing custom scripts to concatenate them. repopack walks your directory tree, respects `.gitignore` and `.repopackignore` rules, and renders every source file into one structured markdown document with a file tree, syntax-highlighted code blocks, and an accurate token count. The output is immediately ready to paste into ChatGPT, Claude, or Gemini — or to pipe into an API call.

## Features

- Generates a single markdown file with a file tree and all source files as fenced code blocks
- Respects `.gitignore` and a project-local `.repopackignore` automatically
- Skips binary files (images, fonts, archives, executables) with no configuration
- Token counting via `cl100k_base` (tiktoken) with per-model context usage breakdown (GPT-4, Claude, Gemini)
- `--compress` strips comments and collapses blank lines to shrink token count by 10–30%
- `--max-tokens` drops the largest files first to fit within a token budget
- `--copy` sends output directly to the clipboard instead of writing a file
- `--stats` prints token usage without writing any output file

## Usage

```bash
# Pack the current directory into repopack-output.md
npx @avinashchby/repopack

# Pack a specific directory with a custom output path
npx @avinashchby/repopack ./src -o context.md

# Strip comments and collapse whitespace before packing
npx @avinashchby/repopack --compress

# Stay within a token budget — largest files are dropped first
npx @avinashchby/repopack --max-tokens 100000

# Include only TypeScript files, exclude tests, and copy to clipboard
npx @avinashchby/repopack --include "**/*.ts,**/*.tsx" --exclude "**/*.test.*" --copy

# Print token stats against model context windows without writing a file
npx @avinashchby/repopack --stats
```

## Example Output

Terminal summary after a successful run:

```
Packed 34 file(s) | ~18,400 tokens → repopack-output.md

Total tokens: 18,400

Model context usage:
  GPT-4      18,400 / 128,000 tokens (14.4%)
  Claude     18,400 / 200,000 tokens (9.2%)
  Gemini     18,400 / 1,000,000 tokens (1.8%)
```

The generated markdown file looks like this:

```markdown
# Repository: my-project
Generated by repopack | 34 files | ~18,400 tokens

## File Tree
\`\`\`
my-project
├── src
│   ├── index.ts
│   └── utils.ts
└── package.json
\`\`\`

## src/index.ts
\`\`\`typescript
// file contents here
\`\`\`
```

